NUMBER GUESSING GAME

Description
This is a Java console application that generates a random number between 1 to 100. The user tries to guess the number. The program provides hints such as "TOO HIGH"
or "TOO LOW" until the correct number is guessed.

How to Run
1. Compile the program : javac Decodelabs_p1.java
2. Run the program : java Decodelabs_p1
